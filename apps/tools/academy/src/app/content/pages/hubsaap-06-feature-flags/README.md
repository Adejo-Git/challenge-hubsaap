# HubSAaP — Feature Flags

Resumo
----
Este diretório contém o material do módulo "Feature Flags" para a Academy HubSAaP. Aqui estão os conceitos, padrões de governança, integração e exemplos de implementação usados pela plataforma.

Páginas incluídas
----
- `hubsaap-06-feature-flags-fundamentos-de-feature-.page.json` — Fundamentos e conceitos de Feature Flags.
- `hubsaap-06-feature-flags-namespace-por-toolkey.page.json` — Estratégia de namespace por `toolKey` para isolar flags entre ferramentas.
- `hubsaap-06-feature-flags-defaults-vs-overrides.page.json` — Comportamento padrão vs sobrescritas (defaults vs overrides).
- `hubsaap-06-feature-flags-integração-com-toolfeat.page.json` — Integração com o serviço/SDK de feature flags (ToolFeat).
- `hubsaap-06-feature-flags-governança-e-auditabili.page.json` — Requisitos de governança, auditoria e rastreabilidade.
- `hubsaap-06-feature-flags-api-reativa-isenabledwa.page.json` — API reativa (`isEnabled`) e exemplos de consumo.

Como contribuir
----
- Edite ou adicione arquivos `.page.json` nesta pasta seguindo o padrão existente.
- Mantenha a documentação concisa e inclua exemplos de uso e contratos quando aplicável.
- Abra um PR com alterações e inclua reviewers das equipes de plataforma e arquitetura quando afetar contratos ou governança.

----
